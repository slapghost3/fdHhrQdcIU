# 学生管理系统简介 python241

## 项目概述

学生管理系统是基于 Python 语言，结合 SQL Server 和 MySQL 数据库，以及 HTML 前端技术构建的一套适用于校园环境的管理系统。该系统旨在提供学生信息、成绩、宿舍等多方面的管理功能，同时支持管理员进行高效管理。

## 技术栈

- **后端**: Python
- **数据库**: SQL Server / MySQL
- **前端**: HTML

## 功能模块

- **学生信息管理**: 支持学生信息的录入、查询、修改和删除。
- **学生成绩管理**: 提供成绩的录入、查询、修改和删除功能。
- **宿舍信息查询**: 学生可查询宿舍分配情况及相关信息。
- **管理员功能**: 管理员可以进行全面的系统管理。

## 系统角色

- **学生**: 查询个人信息、成绩以及宿舍信息。
- **管理员**: 管理学生信息、成绩以及系统设置。

## 运行环境

- **后端**: Python 环境配置
- **数据库**: SQL Server 或 MySQL
- **前端**: HTML5 兼容的浏览器

## 部署步骤

1. 配置数据库环境（SQL Server 或 MySQL）。
2. 部署后端 Python 应用。
3. 部署 HTML 前端文件。

## 目录结构

```
/student_management_system
|-- /backend # 后端代码目录
| |-- /models # 数据库模型
| |-- /views # 视图处理
| |-- /controllers # 控制器逻辑
|-- /database # 数据库相关文件
|-- /frontend # 前端文件目录
| |-- /html # HTML 文件
|-- /static # 静态资源
```
